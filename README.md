# Репозиторий проектов в области машинного обучения

Каталог содержит два проекта:

- **Кредитный скоринг**: система оценки платежеспособности клиентов и рекомендации по сумме кредита.
- **Telegram-бот для прогнозирования акций**: бот, прогнозирующий цену акций на 30 дней, используя несколько моделей машинного обучения.

Каждый проект размещён в отдельной папке и может быть запущен самостоятельно. Они служат образовательными целям и не подходят для прямого использования в коммерческой среде без дополнительной проверки и настройки.

## Структура каталога

bashprojects_catalog/
```
├── credit_scoring_model/      # Кредитный скоринг
└── stock_predict_bot/         # Telegram-бот для прогнозирования акций
```
## Ключевые особенности каждого проекта
### Кредитный скоринг
- Прогнозирует вероятность одобрения кредита и рекомендованную сумму займа.
- Содержит полную инфраструктуру для сбора данных, обучения моделей, мониторинга и автоматизации процесса.
### Telegram-бот для прогнозирования акций
- Загружает историю котировок и строит прогноз на 30 дней.
- Предоставляет рекомендации по покупкам и продажам акций на основе простого анализа прогнозируемых значений.
- Графический отчет и основные метрики предоставляются пользователю.
